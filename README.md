## 🗺️ The Quest Map

Complete quests (issues) in each domain. Close an issue when you've mastered the topic. Each domain ends with a **🏆 Boss Fight** quiz!

## 🎮 How to Play

1. **Pick a quest** — Open an issue from any domain
2. **Study the topic** — Follow the learning objectives and resources in the issue
3. **Complete the challenge** — Do the hands-on task described
4. **Close the issue** — GitHub Actions auto-updates your progress ✨
5. **Boss fights** — Complete the quiz file linked in the boss issue, then close it
6. **Level up** — Watch your XP grow and badges unlock!

### 🔥 Pro Tips

- Work through domains in order, or jump around — your call
- Boss quizzes are in the `quizzes/` folder — honor system!
- The GitHub Actions workflow auto-updates the progress bar on each issue close
- Star ⭐ this repo to bookmark your quest
