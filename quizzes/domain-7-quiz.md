### Q1. Which open-source license is the most permissive, only requiring attribution in copies?
- A) GNU GPL v3
- B) Apache License 2.0
- C) MIT License
- D) AGPL v3

### Q2. What is a key characteristic of the GNU GPL v3 license?
- A) It allows proprietary use with no restrictions
- B) It is a copyleft license that requires derivative works to also be distributed under the GPL
- C) It only applies to documentation
- D) It prohibits commercial use

### Q3. How does the Apache License 2.0 differ from the MIT License?
- A) Apache does not allow commercial use
- B) Apache includes an explicit grant of patent rights and requires stating significant changes; MIT does not
- C) MIT requires patent grants; Apache does not
- D) They are identical in all respects

### Q4. What is InnerSource?
- A) A GitHub-specific programming language
- B) The practice of applying open-source collaboration principles (open contribution, transparency, community) within an organization's private repositories
- C) A type of open-source license
- D) A GitHub Enterprise-only feature for code scanning

### Q5. What is the purpose of a CODE_OF_CONDUCT.md file?
- A) To list the programming languages used in the project
- B) To establish expected behavior standards for community participants, fostering a welcoming and inclusive environment
- C) To define coding style rules
- D) To configure automated testing

### Q6. What is GitHub Sponsors?
- A) A paid advertising program on GitHub
- B) A feature that allows the community to financially support open-source developers and projects directly on GitHub
- C) A venture capital program for startups
- D) A GitHub Enterprise licensing model

### Q7. What is the GitHub Stars program?
- A) The star count on a repository
- B) A recognition program that highlights influential developers, educators, and community leaders who inspire and support others
- C) A rating system for GitHub Apps
- D) A premium tier of GitHub Sponsors

### Q8. Which community health file provides guidance on how to report security vulnerabilities?
- A) CONTRIBUTING.md
- B) CODE_OF_CONDUCT.md
- C) SECURITY.md
- D) SUPPORT.md
